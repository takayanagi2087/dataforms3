プロジェクト初期化
eclipseのプロジェクト内に以下のファイルを作成して、プロジェクトを初期化します。
/src/java/log4j2.xml デフォルトのlog4j2.xmlを作成します。
/src/webapp/WEB-INF/web.xml 必要最小限のweb.xmlを作成します。
/src/webapp/META-INF/context.xml データベースApache Derby用の設定例を作成します。
/src/webapp/WEB-INF/dataforms.conf.jsonc dataforms3.jarの設定ファイルを作成します。
