Create the following files in your Eclipse project to initialize the project.
/src/java/log4j2.xml Create a default log4j2.xml. /src/webapp/WEB-INF/web.xml Create a minimal web.xml.
/src/webapp/META-INF/context.xml Create a sample configuration for the Apache Derby database. /src/webapp/WEB-INF/dataforms.conf.jsonc Create a configuration file for dataforms3.jar.
