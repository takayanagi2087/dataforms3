createSubclass(subClassName, properties, superClass) → {function}

サブクラス作成メソッド。 

SubClass = createSubclass("SubClass", {"p1":"v1", "p2":2}, "SuperClass");

上記呼び出しで、以下のコードを生成して実行します。

let SubClass = class extends SuperClass {
	constructor() {
		super();
		this.p1 = "v1";
		this.p2 = 2;
	}
}
SubClass;
