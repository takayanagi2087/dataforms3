Passkey name
You can register multiple passkeys. Please specify a name to distinguish each passkey.
Register the currently logged in device as a passkey. What is a passkey?...
A passkey is a function that registers your device as a key, and when you use that key, you are required to verify it using the device's authentication functions (fingerprint, face, PIN, etc.).
Using a passkey allows for authentication without a password.
Multiple passkeys can be registered to a user account, which is the same as having a duplicate key.

A recovery code is a code that allows you to log in if multi-factor authentication cannot be used due to hardware failure or other reasons.
