パスキー名称
パスキーは複数登録できます。そのパスキーを区別するための名前を指定してください。
現在ログインしている端末をパスキーとして登録します。
パスキーとはご利用の端末をカギとして登録する機能で、そのカギを使用する際端末の持つ認証機能(指紋、顔、PIN等)での確認が行われます。
PCで登録したパスキーはそのPCでのみ利用することができます。QRコードの読取ができる、iPhone, iPad, Android端末で登録したパスキーは、他の半末から使用できます。
ユーザ情報変更で「パスキーを必ずチェックする」を設定すると、パスキーとして登録された端末の認証なしにログインすることが出来なくなります。
この設定でアカウントの乗っ取りを防ぐことがてきますが、パスキーとして使用した端末が使えなくなるとログインが出来くなります。
そのため複数の端末でパスキーを登録し、そのパスキーが使えることを検証してから「パスキーを必ずチェックする」を有効にしてください。
