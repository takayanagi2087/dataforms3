Register the currently logged in device as a passkey. What is a passkey?...
A passkey is a function that registers your device as a key, and when you use that key, you are required to verify it using the device's authentication functions (fingerprint, face, PIN, etc.).
Using a passkey allows for authentication without a password.
Multiple passkeys can be registered to a user account, which is the same as having a duplicate key.
A passkey registered on a PC can only be used on that PC. A passkey registered on an iPhone, iPad, or Android device that can read QR codes can be used from other devices (PCs, etc.).
You can also share your passkey with devices linked to your Google account or iCloud.
If you set "Always check passkey" in User information update, you will not be able to log in without authenticating the device registered as the passkey.
This setting will help prevent your account from being hijacked, but if the device you used as your passkey becomes unusable, you will not be able to log in.
Therefore, please register a passkey on multiple devices and make sure that you can use the passkey before enabling "Always check passkey."
